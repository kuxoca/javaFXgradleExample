# JavaFX Gradle Example

![](https://shields.io/badge/Java-17-orange) ![](https://shields.io/badge/JavaFX-20-blue)

## Build

```shell
./gradlew clean build
```

## Launch

```shell
java -jar build/libs/HelloFX-0.1-SNAPSHOT-jar-with-dependencies.jar
```
